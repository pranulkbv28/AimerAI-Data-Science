# AimerAI-Data-Science

## Initial Setup

- After cloning the repository, navigate to the project directory and intialise a virtual enviranment:

  - For windows:

    ```bash
    python -m venv .venv
    ```

  - For linux/macOS:

    ```bash
    python3 -m venv .venv
    ```

- Activate the virtual environment:

  - For windows:

    ```bash
    venv\Scripts\activate
    ```

  - For linux/macOS:

    ```bash
    source .venv/bin/activate
    ```

- Install the required dependencies using pip:

  - For windows:

    ```bash
    pip install -r requirements.txt
    ```

  - For linux/macOS:

    ```bash
    pip3 install -r requirements.txt
    ```
